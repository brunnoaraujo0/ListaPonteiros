- Caso x seja declado como um vetor do tipo char:
x + 1 = 4093;

- Caso x seja declado como um vetor do tipo int:
x + 1 = 4094;

- Caso x seja declado como um vetor do tipo float
x + 1 = 4096;
x + 2 = 409A;

- Caso x seja declado como um vetor do tipo double
x + 1 = 409A;
